TLG - The Libre Group web site 2016

Oct 2016
--------

Setup:

- clone this repo (into tlg/ by default)
- cd tlg
- familiarize yourself
- cd conf
- virtualenv env
- pip -r requirements.txt
- cd ..
- ./go.sh (activeates venv)
- ./manage.py <whatever>

That's it!
